# Notes Backend
## Requirements
1. NodeJS 10.15.0
2. MongoDB
## Install instruction
1. Clone repository
2. `npm install`
3. create a .env file with the following variable:
    ```
    PORT=10010
    MONGODB=mongodb://localhost:27017/notes
    ```
4. `npm run mock`
5. `npm start`
6. Navigate to http://localhost:10010/docs for documentation
